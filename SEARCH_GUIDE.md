# 🔍 Guide Complet - Fonctionnalité de Recherche des Salles

## Vue d'ensemble

La fonctionnalité de recherche des salles du Campus Smart Campus permet aux étudiants, enseignants et administrateurs de trouver rapidement une salle sur le campus selon leurs critères spécifiques. C'est une solution complète et intuitive pour l'orientation sur le campus.

---

## ✨ Caractéristiques principales

### 1. **Recherche Simple et Rapide** 🔍
- **Recherche par nom de salle** : Tapez le nom comme "Salle 101" ou "Amphithéâtre"
- **Recherche par numéro** : Recherchez directement le numéro ou l'ID de la salle
- **Recherche par bâtiment** : Trouvez les salles d'un bâtiment spécifique
- **Recherche en temps réel** : Les résultats s'affichent au fur et à mesure de la saisie

### 2. **Filtres Avancés** ⚙️

#### 🏢 **Filtre par Bâtiment**
- Sélectionnez un ou plusieurs bâtiments du campus
- Voir uniquement les salles d'un bâtiment
- Exemple: Sélectionner "Bâtiment A" pour voir toutes les salles de ce bâtiment

#### 👥 **Filtre par Capacité**
- **Capacité Minimum** : Trouvez les salles pouvant accueillir au moins X personnes
- **Capacité Maximum** : Trouvez les salles pouvant accueillir au maximum X personnes
- **Plage de capacité** : Utilisez les deux pour une plage spécifique
- Exemple: Min 20, Max 50 pour une salle moyenne

#### 🎛️ **Filtre par Équipements**
- Sélectionnez les équipements dont vous avez besoin
- Options disponibles:
  - 📹 Vidéoprojecteur / Projecteur
  - 💻 Ordinateur
  - 📶 Connexion Wi-Fi
  - ✏️ Tableau blanc / noir
  - 🔊 Système audio
  - ❄️ Climatisation
  - Et plus...

### 3. **Affichage des Résultats** 📊
- **Nombre de salles trouvées** : Affichage du total des résultats
- **Capacité totale** : Nombre total de places disponibles dans les résultats
- **Filtre chips** : Tags affichant les filtres actifs avec option de suppression
- **Bouton Réinitialiser** : Supprimer tous les filtres en un clic

### 4. **Détails des Salles** 📋
Pour chaque salle trouvée, vous voyez:
- **Nom et localisation** (bâtiment et étage)
- **Capacité** (nombre de personnes)
- **Équipements disponibles** (affichés avec icônes)
- **Statut de disponibilité** (Libre/Occupée)
- **Bouton Favoris** ❤️ pour ajouter aux favoris
- **Accès aux détails complets** en cliquant sur la salle

## 🚀 Guide d'Utilisation

### Étape 1: Accéder à la Recherche
1. Depuis l'accueil, cliquez sur "Liste des salles"
2. Ou depuis le menu, sélectionnez "Salles du Campus"

### Étape 2: Recherche Simple
```
1. Tapez dans la barre de recherche
2. Ex: "Salle 101", "Amphi", "Bâtiment A"
3. Les résultats s'affichent automatiquement
4. Cliquez sur une salle pour voir les détails
```

### Étape 3: Utiliser les Filtres Avancés
```
1. Cliquez sur l'icône ⚙️ (tune) à droite de la barre de recherche
2. Sélectionnez vos critères:
   - Bâtiment désiré
   - Capacité minimum et/ou maximum
   - Équipements nécessaires
3. Cliquez sur "Appliquer"
4. Les résultats sont filtrés automatiquement
```

### Étape 4: Combiner Recherche et Filtres
```
Vous pouvez utiliser la recherche ET les filtres ensemble:
- Tapez "Salle" dans la barre de recherche
- Puis appliquez le filtre "Bâtiment A"
- Résultat: Salles contenant "Salle" dans le Bâtiment A
```

### Étape 5: Modifier les Filtres
```
Les filtres actifs s'affichent comme des tags:
- Cliquez sur la croix (X) d'un tag pour le supprimer
- Cliquez sur "Réinitialiser" pour tous les supprimer
- Cliquez sur ⚙️ pour en ajouter d'autres
```

## 🎯 Conseils et Astuces

### 1. **Recherche Intelligente**
- La recherche ne distingue PAS la casse (Salle = salle)
- Cherchez par nom, numéro, ou bâtiment
- Les résultats incluent la description de la salle

### 2. **Optimiser les Filtres**
- Commencez par un filtre général (bâtiment)
- Affinez avec les équipements ou capacité
- Trop de filtres = moins de résultats

### 3. **Utiliser les Favoris** ❤️
- Cliquez sur le cœur pour ajouter aux favoris
- Accédez facilement à vos salles préférées
- Gestion centralisée dans "Mes favoris"

### 4. **Consulter les Détails Complets**
- Cliquez sur une salle pour voir:
  - Description complète
  - Dates de création/modification
  - Liste complète des équipements
  - Boutons "Voir sur la carte" et "Voir en AR"

## 🔧 Spécifications Techniques

### Architecture
- **Frontend:** Flutter Material Design 3
- **État:** Provider (ChangeNotifier)
- **Base de données:** Firebase Firestore (temps réel)
- **Recherche:** Filtre côté client pour latence zéro

### Méthodes Principales
1. **searchRooms(query)** - Recherche simple
2. **filterByBuilding(building)** - Filtre par bâtiment
3. **filterByEquipment(equipment)** - Filtre par équipement
4. **filterByMinCapacity(capacity)** - Min capacité
5. **filterByMaxCapacity(capacity)** - Max capacité
6. **applyAdvancedFilters()** - Tous les filtres combinés
7. **resetFilters()** - Réinitialiser tous les filtres

### Performance
- Recherche instantanée (< 100ms)
- Support de 1000+ salles
- Filtres combinables sans limite
- Mise à jour en temps réel des données

## ❓ FAQ

### Q1: Comment chercher une salle inexistante?
**R:** Le système affichera "Aucune salle trouvée" avec un message suggérant de modifier les critères.

### Q2: Puis-je réserver une salle depuis la recherche?
**R:** Pas encore, mais cette fonctionnalité est en développement pour les enseignants.

### Q3: Les équipements sont-ils mis à jour automatiquement?
**R:** Oui, la liste des équipements se met à jour quand une salle est modifiée par un administrateur.

### Q4: Comment exporter les résultats?
**R:** Cette fonctionnalité n'est pas encore disponible. Vous pouvez faire une capture d'écran.

### Q5: Y a-t-il un historique de recherche?
**R:** Non, mais vous pouvez ajouter aux favoris pour garder les salles importantes.
